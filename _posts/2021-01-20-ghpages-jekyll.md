拥有自己博客网站的最简便方案：
1. 注册 Github 账号，并在浏览器上登录 Github。
2. 选择网站[模板][gh-themes]，点击进入该模板的仓库页面后，点击右上角`Fork`，打开自己 Github 的`Your repositories`页面，观察是否存在上一步 fork 的仓库，若没有就刷新页面。
